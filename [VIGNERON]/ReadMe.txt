                                                                    Tutoriel
---------------------------------------------------------------------------------------------------------------------------------------------------------

Glisser dans votre le dossier resource de votre serveur


Mettez ceci dans votre servuer.cfg : 

ensure Vigneron

Ajoutez ceci dans votre fichier items d'ox_inventory :

	["raisin"] = {
		label = "Raisin",
		weight = 1,
		stack = true,
		close = true,
	},

	["vin"] = {
		label = "Vin",
		weight = 1,
		stack = true,
		close = true,
	},
